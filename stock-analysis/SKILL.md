# 美股每日分析 Skill

## 🎯 Skill 目的

生成專業、結構化、易讀的美股市場每日分析報告，適合投資者參考和決策。

---

## 📋 使用時機

**觸發條件：**
- 用戶要求「美股分析」、「市場分析」、「今日美股」
- 需要分析三大指數和焦點個股
- 製作每日投資筆記

**範例觸發：**
```
請生成今天的美股分析
分析一下昨天美股的表現
幫我看看今天科技股怎麼樣
```

---

## 📊 報告結構

### 1. 標題區（必須）

```markdown
# 📈 美股市場分析 - YYYY-MM-DD 星期X

> **分析時間：** YYYY-MM-DD HH:mm (台北時間)
> **市場狀態：** [開盤中/已收盤/週末休市]
```

### 2. 市場總覽（必須）

#### 2.1 三大指數表格

| 指數 | 收盤價 | 漲跌點數 | 漲跌幅 | 狀態 |
|------|--------|----------|--------|------|
| 📊 S&P 500 | 6,836.17 | +3.41 | +0.05% | 🟢 |
| 📊 道瓊工業 | 49,500.93 | +48.95 | +0.10% | 🟢 |
| 📊 那斯達克 | 22,546.67 | -49.75 | -0.22% | 🔴 |

**必須包含：**
- 精確到小數點後兩位
- 漲跌用 +/- 表示
- 狀態用 emoji（🟢漲/🔴跌/⬜️平）

#### 2.2 市場情緒（必須）

```markdown
### 市場情緒
- **總體趨勢：** [看漲/看跌/盤整]
- **成交量：** [放量/縮量/持平] 
- **波動度：** VIX [數值] ([漲跌]%)
```

### 3. 焦點個股分析（必須）

#### 3.1 科技股四巨頭（必須全部包含）

```markdown
#### 🍎 蘋果 (AAPL)
- **收盤價：** $XXX.XX
- **漲跌幅：** +/-X.XX%
- **成交量：** X.XX億股
- **分析：** [2-3句話說明漲跌原因]

#### 💻 微軟 (MSFT)
...

#### 🎮 輝達 (NVDA)
...

#### ⚡ 特斯拉 (TSLA)
...
```

#### 3.2 其他重點（至少4檔）

- 📦 亞馬遜 (AMZN)
- 🔍 Google (GOOGL)  
- 📱 Meta (META)
- 🏭 台積電ADR (TSM)

### 4. 重要新聞（必須）

**分三類整理：**

```markdown
### 經濟數據
- [CPI、就業、GDP 等]

### 企業動態  
- [財報、併購、產品發布]

### 政策影響
- [Fed 決策、政府政策]
```

### 5. 產業表現（必須）

| 產業 | 表現 | 領漲股 | 說明 |
|------|------|--------|------|
| 科技 | ✅ +1.2% | NVDA | AI 需求強勁 |
| 金融 | ❌ -0.5% | JPM | 利率擔憂 |
| 能源 | ⬜️ +0.1% | XOM | 油價持平 |

**至少包含5個產業：**
- 科技
- 金融
- 能源  
- 醫療
- 消費

### 6. 市場分析（必須）

```markdown
## 💭 市場分析與觀點

### 今日重點摘要
1. [第一個重點]
2. [第二個重點]
3. [第三個重點]
4. [第四個重點]
5. [第五個重點]

### 技術面觀察
- **S&P 500 支撐位：** 6,800
- **S&P 500 壓力位：** 6,900  
- **技術指標：** RSI 55 (中性)

### 基本面觀察
[2-3段分析當前市場基本面]
```

### 7. 明日展望（必須）

```markdown
## 🎯 明日展望

### 需關注事項
- [ ] [重要財報發布]
- [ ] [經濟數據公佈]
- [ ] [Fed 會議或談話]

### 操作建議
[謹慎/觀望/積極] - [說明理由1-2句]

**風險提示：** [提醒可能的風險]
```

## 🎨 格式規範

### Emoji 標準

**市場方向：**
- 📈 上漲/看多
- 📉 下跌/看空
- 🟢 正面
- 🔴 負面
- ⬜️ 中性/持平

**內容分類：**
- 🎯 重點
- 💡 洞察
- ⚠️ 警告
- 📊 數據
- 📰 新聞
- 🔥 熱點
- ✅ 利多
- ❌ 利空

### 數字格式

```
價格：6,836.17（千位加逗號）
百分比：+0.05%（正負號 + %）
成交量：1.23億股
金額：$1,234.56
```

### 文字格式

**粗體：**
- 股票代碼
- 重要數據
- 關鍵結論

**引用框：**
```markdown
> **核心結論：** [重要觀點]
```

**清單：**
- 最多3層
- 使用 - 或數字
- 每點簡潔

## 🔍 數據收集規範

### 必須使用 web_search

**搜尋策略：**

```
第1次搜尋（指數）：
"S&P 500 Dow Jones Nasdaq closing price today February [日期]"

第2次搜尋（個股）：
"AAPL MSFT NVDA TSLA stock price today"

第3次搜尋（新聞）：  
"US stock market news today [日期]"

第4-5次（深度）：
根據發現的重要新聞深入搜尋
```

### 數據驗證

✅ **必須做：**
- 交叉驗證重要數字
- 標註資料來源
- 註明數據截止時間

❌ **禁止：**
- 猜測數據
- 使用過時資訊
- 混淆不同時間點的數據

### 引用格式

```markdown
根據 Bloomberg 報導，[內容]
來源：CNBC, 2026-02-17
```

## 💬 語氣與風格

### 保持專業客觀

✅ **正確：**
```
數據顯示市場呈現謹慎態度
分析師普遍認為...
技術面顯示...
```

❌ **錯誤：**
```  
股市肯定會漲
一定要買XX股票
絕對會跌
```

### 避免絕對化

**改用相對詞：**
- 「傾向」取代「一定」
- 「可能」取代「必然」
- 「預期」取代「確定」

### 專業術語使用

**鼓勵使用：**
- 技術指標（RSI, MACD, KD）
- 支撐/壓力位
- 產業輪動
- 風險偏好
- 多頭/空頭排列

---

## ⚠️ 重要注意事項

### 1. 免責聲明（絕對必須）

```markdown
---

**免責聲明：** 本分析僅供參考，不構成投資建議。投資有風險，請謹慎評估。
```

### 2. 數據準確性

- 所有數字來自可靠來源
- 衝突時選權威來源
- 不確定標註「待確認」

### 3. 時效性標註

- 明確標註分析時間
- 註明數據截止時間
- 提醒市場持續變化

### 4. 避免具體操作建議

✅ **允許：**
```
技術面顯示需關注 6,800 支撐
建議持續觀察 Fed 政策
風險偏好者可考慮...
```

❌ **禁止：**
```
立即買進 AAPL
快賣掉所有股票  
保證獲利
```
