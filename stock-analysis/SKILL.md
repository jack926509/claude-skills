# 美股分析專業 Skill

## 目的
協助 Claude 生成專業、結構化的美股市場分析報告。

## 使用時機
- 用戶要求生成美股分析報告
- 需要分析市場數據
- 製作投資簡報

## 報告結構

### 1. 市場總覽
- 三大指數（S&P 500、道瓊、那斯達克）
- 漲跌幅度和原因分析
- 市場情緒評估

### 2. 焦點個股
- AAPL、MSFT、NVDA、TSLA 等
- 股價變動
- 重要新聞和事件

### 3. 產業分析
- 科技、金融、能源等主要產業
- 產業輪動觀察

### 4. 市場展望
- 技術面分析
- 基本面觀察
- 操作建議

## 格式要求

### 使用 Emoji
- 📈 上漲
- 📉 下跌
- 🎯 重點
- 💡 洞察
- ⚠️ 警告

### 表格格式
使用 Markdown 表格呈現數據：

| 指數 | 收盤價 | 漲跌幅 | 狀態 |
|------|--------|--------|------|
| S&P 500 | 數據 | % | emoji |

### 重點標記
- 使用粗體標記重要資訊
- 使用引用框標記關鍵結論
- 使用清單組織資訊

## 資料來源要求
- 使用 web_search 工具收集最新數據
- 標註資料來源和時間
- 交叉驗證重要數據

## 注意事項
- 保持客觀中立
- 數據必須準確
- 避免投資建議（僅供參考）
- 加上免責聲明
