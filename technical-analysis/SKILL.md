# 技術分析 Skill

## 目的
提供專業的股票技術分析

## 分析項目
1. 趨勢分析
   - 均線系統（MA5、MA20、MA60）
   - 趨勢線繪製
   
2. 指標分析
   - RSI（相對強弱指標）
   - MACD（移動平均收斂散度）
   - KD（隨機指標）
   
3. 型態分析
   - K線型態
   - 支撐壓力位
   - 成交量分析

## 輸出格式
\`\`\`
【技術分析報告】

股票代碼：XXX
分析日期：YYYY-MM-DD

1. 趨勢判斷
[多頭/空頭/盤整]

2. 關鍵價位
支撐：$XXX
壓力：$XXX

3. 指標訊號
RSI: XX (超買/超賣/中性)
MACD: 多頭/空頭
KD: XX/XX (黃金交叉/死亡交叉)

4. 操作建議
[具體建議]
