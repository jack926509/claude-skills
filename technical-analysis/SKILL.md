# 🔬 技術分析 Skill v2.0

## 🎯 Skill 目的
提供個股或指數的深度技術面分析，協助投資者判斷趨勢、尋找關鍵價位，並整合多重指標給出綜合評估。

---

## 📋 觸發條件

```
「分析 [股票代碼] 技術面」
「[NVDA/AAPL/S&P500] 現在可以買嗎」
「幫我看 [股票] 的支撐壓力」
「[股票] 的 RSI / MACD / KD 訊號」
```

---

## 🔍 數據收集流程

### Step 1：取得近期價格
```
"[股票代碼] stock price chart technical analysis [月份] 2026"
```

### Step 2：查詢指標數值
```
"[股票代碼] RSI MACD moving average [日期] 2026"
```

### Step 3：查詢近期新聞（補充背景）
```
"[股票代碼] news catalyst [日期] 2026"
```

> 指標數值如無法確認，標注「待確認」，勿自行估算。

### 1. 基本資訊

```markdown
**分析日期：** YYYY-MM-DD

**分析週期：** 日線 / 週線 / 月線
```

### 2. 趨勢判斷（必填）

#### 均線系統

| 均線 | 數值 | 與現價關係 | 意義 |
|------|------|-----------|------|
| MA5（5日）| $XXX | 股價在上/下 | 短期方向 |
| MA20（月線）| $XXX | 股價在上/下 | 中期趨勢 |
| MA60（季線）| $XXX | 股價在上/下 | 長期趨勢 |
| MA200（年線）| $XXX | 股價在上/下 | 牛熊判斷 |

**均線排列：**
- 🟢 多頭排列（MA5 > MA20 > MA60）：趨勢看漲
- 🔴 空頭排列（MA5 < MA20 < MA60）：趨勢看跌
- ⬜️ 糾纏整理：等待方向確認

**整體趨勢判斷：** 多頭 / 空頭 / 盤整

---

### 3. 關鍵價位（必填）

```markdown
| 類型 | 價位 | 說明 |
|------|------|------|
| 強壓力 | $XXX.XX | [原因：前高/整數關卡/均線] |
| 近壓力 | $XXX.XX | [原因] |
| 近支撐 | $XXX.XX | [原因：前低/均線/整數] |
| 強支撐 | $XXX.XX | [原因] |
| 止損建議 | $XXX.XX | 跌破此位考慮出場 |
```

> 支撐/壓力的設定來源：前期高低點、均線、費氏回撤位（23.6%/38.2%/61.8%）

### 4. 技術指標訊號（必填）

#### RSI（相對強弱指數，14日）

| RSI 數值 | 狀態 | 意義 |
|---------|------|------|
| > 70 | 超買 ⚠️ | 短期可能回落 |
| 50-70 | 強勢 🟢 | 多方佔優 |
| 30-50 | 弱勢 🔴 | 空方佔優 |
| < 30 | 超賣 💡 | 反彈機會增加 |

**當前 RSI：** XX（[狀態]）

### 5. K 線型態分析

```markdown
#### 近期重要型態
- **[型態名稱]：** 出現位置 → 意義（看漲/看跌/反轉）
- 例：「十字星」出現在壓力位 → 多空拉鋸，需觀察後續確認

#### 成交量分析
- 近期量能：放大/縮小/持平
- 量價關係：價漲量增（健康）/ 價漲量縮（注意）/ 價跌量增（危險）
```

---

## ⚠️ 重要注意事項

1. **技術分析是概率工具，非預測工具**，所有結論均為「傾向」而非「確定」
2. 多空訊號衝突時，以**較長週期**為主要判斷依據
3. 指標純供參考，重要進出場決策需同步考量**基本面與市場大環境**
4. 不得出現「一定漲」「保證獲利」等絕對化語言
