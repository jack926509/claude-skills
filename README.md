## ⚡ 快速觸發範例

### 每日例行（推薦指令）

```
請使用我的 stock-analysis + report-writing skill，
生成今天的美股日報（含三大指數、科技七巨頭、
產業輪動、重要新聞與後市展望）
```

### 個股技術分析

```
請使用我的 technical-analysis skill，
分析 NVDA 的技術面現況，包含支撐壓力、RSI、MACD 訊號
```

### 週度策略報告

```
請使用我的 stock-analysis + report-writing skill，
整理本週（2/10-2/14）美股週報，重點分析 AI 衝擊主題
```

### 深度個股報告

```
請使用我的 technical-analysis + report-writing skill，
生成 NVDA 的深度分析報告，包含基本面、技術面與操作建議
```

## 📅 建議每日使用流程

```
08:30（台北時間）— 美股昨晚收盤後
  └─ 使用 stock-analysis skill 生成日報
  └─ 搭配 report-writing skill 輸出格式

09:00（可選）
  └─ 針對當日焦點個股使用 technical-analysis skill
```
